Web-based steganography tool.

This website provides a simple tool for encoding and decoding messages with images.

For Encoding: Go to the Encode section of the website, select an image that you want to encode and enter your super-secret message in the text box below.

For Decoding: Go to the Decode section of the website, select an image that you want to decode and then your super-secret message will appear in text box below.

You can also access the last decoded message and the last encoded image with the links provided at the right of this text.

More information about the project, how the steganography was done and about this website can be found in the about section.
